## 📐 Image Specifications

### Hero Background
| Property | Value |
|----------|-------|
| File | `hero-bg.jpg` |
| Dimensions | 1920 x 1080 px (16:9) |
| Format | JPEG (WebP alternative) |
| Max Size | 500KB |
| Style | Abstract tech/AI visualization, dark theme |
| Notes | Should work with dark gradient overlay |

**Suggested Content:**
- Neural network visualization
- Abstract data streams
- Geometric patterns with tech feel
- Blue/purple color scheme

---

## 🎨 Brand Guidelines

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Purple | `#6366f1` | CTAs, highlights |
| Secondary Violet | `#8b5cf6` | Gradients |
| Accent Cyan | `#06b6d4` | Accents, links |
| Dark Background | `#0f172a` | Backgrounds |
| Text Gray | `#94a3b8` | Body text |

### Style Guidelines
- **Tone:** Professional, modern, tech-forward
- **Theme:** Dark mode optimized
- **Imagery:** Abstract tech, clean, minimal
- **Avoid:** Stock photos that look generic, overly busy images

## 🛠️ Image Optimization Tools

### Recommended Tools
1. **Squoosh** (https://squoosh.app/) - Browser-based
2. **ImageOptim** (Mac) - Desktop app
3. **TinyPNG** (https://tinypng.com/) - PNG/JPEG compression
4. **SVGOMG** (https://jakearchibald.github.io/svgomg/) - SVG optimization

### Optimization Settings
- **JPEG Quality:** 80-85%
- **WebP Quality:** 75-80%
- **PNG:** Use for logos/icons only
